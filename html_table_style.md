# html的table属性介绍
* [tr/th/td 区分](https://blog.csdn.net/mxk4869/article/details/81067106)
```
具体格式是:(两行两列) 
<table> 
<tr> 
<td></td> 
<td></td> 
</tr> 
<tr> 
<td></td> 
<td></td> 
</tr> 
</teble>
```
```
<table>是<tr>的上层标签
<tr>必须在一个<table></table>里面,它不能单独使用,相当于<table>的属性标签. 
<table>标示一个表格,<tr>标示这个表格中间的一个行 
<td>标示行中的一个列,需要嵌套在<tr></tr>中间

<th>和<td>的区别是 <th>标签的字体是粗体
```
